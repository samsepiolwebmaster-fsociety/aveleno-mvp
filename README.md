# AVELENO (HARENBI)

Distribution MP3 directe pour artistes indépendants francophones.

**Live:** https://harenbi.netlify.app

## Concept
Plateforme de vente directe de musique — l'artiste vend ses morceaux et licences directement à ses fans, sans intermédiaire.

- Artiste garde 100% du contrôle sur les prix et licences
- Paiement Stripe → téléchargement immédiat
- Player 30s intégré

## Stack
- Frontend: HTML pur
- Auth: Supabase
- Database: Supabase PostgreSQL
- Storage: Supabase Storage
- Paiement: Stripe
- Functions: Netlify Functions
- Hosting: Netlify
